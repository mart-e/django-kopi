What is Kopi ?

Kopi is an open source blogging software in Python using the framework Django. All the code can be found on Gitorious, feel free to fork it but mostly to improve it.

Why not Wordpress ?

I have been using Wordpress for a long time. It was ok but I feld like using a tank. Also there was some tweaking I wanted to do and looking at that wordpress code made me cry blood (did I told you I don't like PHP ?). I love Django and thaught that with a few lines of code and hours, I could have something that suits all my needs. I admit, it was more than just a few lines and hours.

Why named Kopi ?

Kopi is coffee in Indonesian, as I will certainly die of a hart attack drinking too much of it, I think it deservered it. Also "Kopi" sounds like "copy", which you can all do about my articles and code. Free licences inside.
