What is Kopi ?

Kopi is an open source blogging software in Python using the framework Django. All the code can be found on Gitorious, feel free to fork it or improve it.

Why named Kopi ?

Kopi is coffee in Indonesian, as I will certainly die of a hart attack drinking too much of it, I think it deservered it. Also "Kopi" sounds like "copy", which you can all do about my articles and code. Free licences inside.

Why not Wordpress ?

I have been using Wordpress for a long time. It was ok but I felt like using a tank, I used maybe 2% of what it was capable of. Also there was some tweaking I wanted to do and looking at that wordpress code made me cry blood (did I told you I don't like PHP ?). I love Django and thaught that with a few lines of code and hours, I could have something that suits all my needs. I admit, it was more than just a few lines and hours.

Why not use another Django blog app ?

There exists many Django blog applications but none are really out of the mass or are way too complexe. To match my exact needs, I prefered create my own blog application (and it was fun and a good learning excercice).

You wrote all the code ?

Not exactly. To create this project, I took parts from the Django basic apps that I modified and updated to use the latest functionalities of Django. . The inlines application comes from the fork of Peter Hogg. Thank you to all of them.
